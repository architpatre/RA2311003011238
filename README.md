# RA2311003011238 — Frontend Quiz Repo

This repository is structured to match typical quiz submission rules:

- **Single repo, single branch**
- **Separate folders per attempted question**
- **No online IDEs**
- **Keep `node_modules/` and OS junk out of git**

## Folder structure

- `frontend/q1/` — Question 1 (starter)
- `frontend/q2/` — Question 2 (starter)
- `frontend/q3/` — Question 3 (starter)

Create more folders as needed: `frontend/q4/`, etc.

## How to run (no dependencies)

Each question folder is a plain HTML/CSS/JS project. You can open `index.html` directly, or use a local server:

```powershell
cd frontend/q1
npx serve .
```

## GitHub push checklist

- Repo name is your roll/reg no: `RA2311003011238`
- All work in the same branch (usually `main`)
- One folder per question attempted
- `.gitignore` includes `node_modules/`, `.DS_Store`, etc.
